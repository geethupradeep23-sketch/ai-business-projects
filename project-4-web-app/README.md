# Project 4: Resume/Cover Letter Generator 📄

## What it does
A simple web app that generates professional resumes and cover letters based on user input. Users can customize and download their documents.

## Use Cases
- **Job Seekers:** Generate tailored resumes for each job
- **Career Changers:** Create new resumes for different industries
- **Students:** Generate professional CVs for internships
- **Freelancers:** Create professional service descriptions

## How to Run

```bash
cd project-4-web-app
pip install -r requirements.txt
python app.py
```

Then open your browser to `http://localhost:5003`

## Code Explanation

1. **User Form:** Collects job info, skills, experience
2. **AI Generation:** Uses OpenAI to generate tailored documents
3. **Preview:** Shows the generated content before download
4. **Download:** Users can download as PDF or text

## How It Works

1. Fill in the job application form
2. Choose resume or cover letter
3. AI generates tailored document
4. Download and use it
